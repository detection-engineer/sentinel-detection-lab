# sentinel-detection-lab
Azure Sentinel detection engineering and cloud security use cases
This repository demonstrates hands-on detection engineering and security operations work using Microsoft Sentinel, Microsoft Entra ID telemetry, and endpoint logs.

# Objectives
 1. Design and tune SIEM detections for identity and endpoint threats
 2. Apply MITRE ATT&CK mapping to detection logic
 3. Simulate real-world attack scenarios in a controlled lab
 4. Explore detection-as-code concepts using Sigma and Python

# Technologies
 1. Microsoft Sentinel
 2. Microsoft Entra ID Sign-in & Audit Logs
 3. KQL (Kusto Query Language)
 4. Sigma Rules
 5. Python (Detection-as-Code)
 6. Sysmon (Endpoint Telemetry)
